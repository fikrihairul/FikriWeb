Tugas Pengantar Teknologi dan Komunikasi

Sebelum itu, saya ingin mengucapkan terimakasih kepada bapak(siapa) yang telah membimbing mata kuliah PTIK ini. Saya membuat tugas berupa website dengan tambahan efek animasi parallax scroll dibantu dengan referensi dibeberapa platform seperti youtube dan artikel di google. Mohon maaf jika tugas yang saya berikan ini masih memiliki beberapa kesalahan, karena masih ditahap belajar. Oleh Karena itu, saya meminta kritik dan sarannya kepada bapak(siapa) agar bisa meningkatkan wawasan terkait website ini.

Hambatan dalam mengerjakan tugas ini adalah keterbatasan device dan mengharuskan untuk meminjam laptop teman sehingga mampu mengerjakan tugas ini h-1 sebelum deadline, lalu selain dari device ada juga terkait kurangnya wawasan tentang programming karena lulusan dari sekolah menegah atas dan mengharuskan untuk mempelajari secara mandiri mengenai dasar-dasar tentang html dan css.

Lesson learn yang saya dapatkan pada tugas kali ini adalah saya mendapatkan wawasn baru terkait html, css dan javascript, mengerti perbedaan fungsi dari setiap bahasa pemrograman contohnya pada ddp yang mempelajari c dan ptik yang mempelajari html dan css, menjadi lebih teliti ketika menemukan error pada code dan mengerti setiap syntax itu bisa diapahami serta dianalogikan. Saya ada ketertarikan untuk mendalami ini, karena saya menyukai yang hal dalam mendesain suatu produk seperti poster, animasi, interior, website, aplication, dll sehingga memiliki tujuan untuk menjadi frontend developer.
